RS_Identifier - Identifier used for reference systems.

1

authority

0..1

Citation to person or party responsible for maintenance of the code value.

2

code

1

The alphanumeric value that uniquely identifies the referenced resource. When the identifer has a permanent URL, then use the gmx:Anchor field instead of gco:CharacterString field. Example: urn:ogc:def:crs:EPSG::4326

3

codeSpace

0..1

Namespace of the system in which the code is valid.

4

version

0..1

The version of the code value. Example: 6.18.3
